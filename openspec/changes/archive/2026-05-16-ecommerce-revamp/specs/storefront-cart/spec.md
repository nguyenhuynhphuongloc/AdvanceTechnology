## ADDED Requirements

### Requirement: Responsive Cart Layout
The cart page MUST use a responsive layout that ensures the Order Summary is visually cohesive with the cart items on large screens, and stacks appropriately on mobile devices.

#### Scenario: Viewing cart on desktop
- **WHEN** a user views the cart on a large screen
- **THEN** the cart items and order summary are displayed side-by-side with an appropriate maximum width to prevent excessive spacing
